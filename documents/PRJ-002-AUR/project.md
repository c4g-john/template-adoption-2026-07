## Overview

Aurora is the sample customer-onboarding project this test repository uses to
exercise the adopter path. It exists to prove that a fresh template copy
reaches a green gate and a live dashboard using only the README. Aurora is
fictional and disclosed as such.

validated on every pull request. Out of scope: everything else; this project
carries no delivery commitment.
